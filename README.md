# 🔄 Sincronizador de Archivos (Origen → Destino)

Herramienta en Python que sincroniza archivos desde una carpeta de origen hacia una carpeta de destino, copiando solo los archivos que existen en el destino y evitando crear estructuras innecesarias.

---

## 🚀 Características

- Copia archivos manteniendo la estructura de carpetas
- Solo copia si la carpeta destino existe
- Evita errores al no crear directorios automáticamente
- Permite identificar archivos faltantes
- Útil para backups selectivos o sincronización controlada

---

## 🛠️ Requisitos

- Python 3.x
